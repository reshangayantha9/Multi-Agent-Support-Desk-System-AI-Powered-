# Data Retention & Privacy

**doc_id: KB-005**

## Data We Collect

- **Account data**: name, email, password (hashed), billing info
- **Usage data**: API call logs, feature usage, session metadata
- **Content data**: Any files or data you upload/create within the platform

## Retention Periods

| Data Type | Retention Period |
|-----------|----------------|
| Account data | Duration of account + 90 days after deletion |
| API logs | 90 days |
| Billing records | 7 years (legal requirement) |
| Content data | Duration of account + 30 days |
| Backups | 30 days rolling |

## Data Deletion Requests

To request deletion of your data:
1. Go to **Settings → Privacy → Request Data Deletion**.
2. Submit the form — you'll receive a confirmation email within 24 hours.
3. Deletion is completed within **30 days**.

Note: Billing records are retained for 7 years regardless of deletion requests due to legal obligations.

## GDPR & CCPA Compliance

- We are GDPR-compliant (EU users).
- We honor CCPA requests (California users).
- Data Processing Agreements (DPA) available for Enterprise customers.

## Data Export

Export all your data at any time via **Settings → Privacy → Export My Data**. You'll receive a download link within 24 hours.

## Third-Party Sharing

We do not sell your data. We share data only with:
- Payment processors (Stripe) for billing
- Cloud infrastructure providers (AWS) for hosting
- Analytics tools (anonymized, aggregate only)
